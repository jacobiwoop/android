# Features LuminaZen Marketplace

## Core Features

- [x] **Système de Navigation Dynamique** : Router Vanilla JS avec transitions.
- [x] **Composants Réutilisables** : Header, BottomNav, AdCard, CategoryBar.
- [x] **Dépôt d'Annonces** : Formulaire avec sélection de photos.
- [x] **Messagerie Instantanée** : Communication temps réel via Socket.io.
- [x] **Authentification** : Inscription et connexion avec JWT (Backend).

## Fonctions Natives (Capacitor)

- [x] **Caméra** : Prendre ou choisir des photos pour les annonces.
- [x] **Géolocalisation** : Détecter automatiquement la position de l'utilisateur.
- [x] **Haptique** : Retours physiques lors des actions utilisateur.

## UI/UX

- [x] **Tailwind CSS 4** : Design moderne et ultra-performant.
- [x] **Glassmorphism** : Effets de transparence "Premium".
- [x] **Animations** : Transitions de pages et micro-interactions.
